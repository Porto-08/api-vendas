# API Vendas
Projeto para fins de estudos de NodeJs, Typescript e SOLID. 

# Features
- CRUD de usuários
- Redinifição de Senha com token
- CRUD de produtos
- CRUD de clientes
- Criação, Listagem e Vizualização de pedidos
